### Print Working Directory
```
pwd
```
### List files and dir
```
ls
ls -a
```
### Change dir
```
cd .
cd ..
cd <where you want to go>
```
### Make a new dir
```
mkdir <name of dir to make>
```
### Remove a dir
```
rmdir <Name of dir to delete>
```
### Create copy of file
```
cp <source file> <new location>
```
### Moves file
```
mv <source file> <new location>
```
